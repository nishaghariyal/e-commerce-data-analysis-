# 🛒 E-Commerce Product Analysis (SQL + Power BI)

## 📌 Project Overview

This project focuses on analyzing an e-commerce dataset to understand customer behavior, product performance, and overall business trends. The goal is to transform raw transactional data into actionable insights using SQL and visualize them through an interactive Power BI dashboard.

---

## 🎯 Objectives

* Analyze customer purchasing behavior
* Identify top-performing products and regions
* Measure Customer Lifetime Value (CLV)
* Evaluate order status and operational performance
* Understand customer satisfaction through reviews

---

## 🛠 Tools & Technologies

* SQL Server (Data extraction & transformation)
* Power BI (Dashboard & visualization)
* DAX (Measures & calculations)

---

## 🗂 Dataset

* Brazilian E-Commerce Public Dataset
* Dataset link- https://drive.google.com/drive/folders/1J_gz5rgFwIM-BChVEpLkmEO3LjVpgLV-?usp=drive_link
* Contains data on customers, orders, products, payments, and reviews

---

## 🔍 Key Analysis Performed

### 📊 1. Revenue Analysis

* Total revenue calculation
* Monthly revenue trends

### 👤 2. Customer Analysis

* Customer Lifetime Value (CLV)
* Top customers by spending
* Customer distribution by location

### 📦 3. Product Analysis

* Top-selling products
* Product demand trends

### 🚚 4. Order Analysis

* Order status distribution
* Delivery performance insights

### ⭐ 5. Review Analysis

* Average review score
* Rating distribution (1–5 stars)

---

## 📈 Dashboard Features

* Interactive filters (Date, City, Order Status)
* KPI cards (Revenue, Orders, Customers, Rating)
* Revenue trend visualization
* Customer and product performance charts

---

## 💡 Key Insights

* Identified high-value customers contributing most revenue
* Observed trends in order delivery and cancellations
* Highlighted top-performing products and regions
* Analyzed customer satisfaction using review data

---

## 🚀 Conclusion

This project demonstrates how SQL and Power BI can be used together to perform end-to-end data analysis and generate meaningful business insights from real-world datasets.
